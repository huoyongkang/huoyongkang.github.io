给类绑定属性或方法

我们可以直接在实例或类上，给类绑定属性，如果在实例上绑定，则只对当前实例有效，对于其他实例就会失效，如果要对类生效，就要直接对类进行绑定。

同时，也可以对类绑定方法。

def set_time(h,m,s):
    self.hour = h
    self.minute = m
    self.second = s
    
from types import MethodType
time.set_time = MethodType(set_time,None,time)
t = time()
t.set_time = MethodType(set_time,t,time)

以上就是对类和实例绑定方法的操作，但此时的问题时，如何对这种随意绑定的行为加以限制。

解决方法是利用__slots__参数，在类的定义中直接利用

1

__slots__ = ('minute','hour','second')

就可以 对类的属性加以限制。

在类中定义装饰器，有很多用处，总结如下

  • 对属性定义的范围加以限制
  • 控制只读属性

class Student(object):
    @property
    def score(self):
        return self._score
        
    @score.setter
    def score(self,value):
        if not isinstance(value,int):
            raise ValueError('没有输入整数')
        if value < 0 or value > 100
            raise ValueError('分数输入范围出错')
        self._score = value

__getitem__

类方法__getitem__可以让类能够像list一样取元素

class Fib(object):
    def __getitem__(self,n):
        a,b = 1,1
        for x in range(n):
            a,b = b,a+b
        return a

 def __getitem__(self, n):
        if isinstance(n, int):
            a, b = 1, 1
            for x in range(n):
                a, b = b, a + b
            return a
        if isinstance(n, slice):
            start = n.start
            stop = n.stop
            a, b = 1, 1
            L = []
            for x in range(stop):
                if x >= start:
                    L.append(a)
                a, b = b, a + b
            return L

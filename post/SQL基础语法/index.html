基本概念

数据库：容器 表：某种特定数据类型的结构化清单 表名唯一 模式：关于数据库和表的布局及特性的信息 数据类型：每列所允许的数据类型 行：表中的一个记录 主键：固定的某一列，可以唯一的标识表中的每一行

组合where

1
2
3

from Products

1
2
3

select prod_id, prod_price, prod_name
from Products
where vend_id = 'DLL01' or prod_id <= 4;

组合where的求值顺序

1
2
3
4

select prod_name, prod_price
from products
where (vend_id = 'DLL01' or vend_id = 'BRS01')
        AND prod_price >= 10

日期和时间处理函数

1
2
3

select order_num
from orders
where datepart(yy, order_date) = 2012;

在oracle中没有datepart函数，可用几个完成相同函数完成检索

1
2
3

select order_num
from orders
where to_number(to_char(order_date, 'YYYY')) = 2012;

oracle中另一种方法是使用between函数

1
2
3
4

select order_num
from orders
where order_date between to_date('01-01-2012')
and to_date('12-31-2012');

聚集函数

函数 说明
